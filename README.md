# 💸 Expense Tracker API

Built with Node.js + Express + MongoDB

---
## What This Project Does

A backend REST API to track your daily expenses. You can add, view, update, delete, and filter expenses by category. All data is permanently stored in MongoDB.

---

## Tech Stack


- **Node.js + Express** — Backend server
- **MongoDB Atlas**  — Cloud database
- **Mongoose** — ORM
-**dotenv** — Environment config
